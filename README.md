# 🌐 HTML (HyperText Markup Language)

HTML is the standard markup language used to create web pages. It structures the content on the web and works hand-in-hand with CSS and JavaScript to build responsive, interactive sites.

---

## 🔌 Recommended Extensions

To enhance your HTML development experience, consider installing the following VS Code extensions:

1. **Prettier** – Code formatter for clean and consistent HTML.
2. **Live Preview** – Quickly preview your HTML in a side panel.
3. **Live Server** – Launch a local development server with real-time reload.

---

## 🧱 1. Basic HTML Structure

Every HTML document begins with a standard boilerplate that defines how the browser should interpret the content.

### 🔹 Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
```

🧩 Key Elements

```html<!DOCTYPE html>``` : Declares that the document is using HTML5.

```html<html>``` : The root element that wraps the entire HTML document.

```html<head>``` : Contains meta-information like the page title, links to stylesheets, and scripts.

```html<title>``` : Specifies the title of the web page (shown in the browser tab).

```html<body>``` : Contains all the visible content shown to the user (text, images, links, etc.).

## ✍️ 2. Text Elements & Headings

### Common Tags:
1. Headings: ```<h1> to <h6>```
2. Paragraph: ```<p>```
3. Line break: ```<br>```
4. Horizontal rule: ```<hr>```
### Emphasis Tags:
1. Bold: ```<strong> or <b>```
2. Italic: ```<em> or <i>```

---

## 🔗 3. Links & Navigation

```html<a href="https://example.com">Visit Example</a>```

### Attributes:

```href``` : Destination URL

```target="_blank"``` : Opens link in new tab

---

## 🖼️ 4. Images

```<img src="photo.jpg" alt="A scenic view" />```
### Attributes:

```src``` : Image path

```alt``` : Alternate text

```width, height``` : Optional sizing

---

## 📋 5. Lists

### Types of Lists:

Unordered List: ```<ul>, <li>```

Ordered List: ```<ol>, <li>```

Definition List: ```<dl>, <dt>, <dd>```

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
```

## 📊 6. Tables

```html
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>30</td>
  </tr>
</table>
```
Table Tags:```<table>, <tr>, <th>, <td>```
  
Optional: ```<thead>, <tbody>, <tfoot>, <caption>```

## 🧾 7. Forms & Inputs

```html
<form action="/submit">
  <label for="name">Name:</label>
  <input type="text" id="name" />
  <button type="submit">Send</button>
</form>
```

### Common Input Types:

```text, email, password, radio, checkbox, submit```

### Other Elements:

```<textarea>```
  
```<select>, <option>```

```<fieldset>, <legend>```

---

## 🎥 8. Multimedia

```html
<video controls>
  <source src="video.mp4" type="video/mp4" />
</video>

<audio controls>
  <source src="song.mp3" type="audio/mpeg" />
</audio>
```

### Attributes:

```controls``` : Displays playback controls

```autoplay``` : Starts playing automatically

```loop``` : Repeats media

```muted``` : Starts muted

---

### 🧩 9. Semantic HTML

Semantic elements describe the content they contain. They improve accessibility and SEO.

### Examples:

```<header>``` – Introductory content

```<footer>``` – Closing content

```<nav>``` – Navigation menu

```<main>``` – Main content area

```<section>``` – Thematic grouping

```<article>``` – Independent content

```<aside>``` – Sidebar or extra content

```<figure>/<figcaption>``` – Media and captions

```<time>``` – Timestamps

```<mark>``` – Highlighted text

---

### ♿ 10. Accessibility & Best Practices

Use alt text for all images

Maintain a proper heading structure

Associate inputs with elements

Use semantic tags wherever possible
